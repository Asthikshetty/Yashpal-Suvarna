# Yashpal Anand Suvarna Official Website Concept

Premium React + Vite + TypeScript website concept for Yashpal Anand Suvarna, MLA of Udupi Constituency.

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Included Features

- Premium saffron, navy, white, and gold leadership design
- Responsive hero, biography, timeline, development dashboard, and constituency map
- Citizen grievance form and request tracking concept
- News, gallery, video, reports, event/newsletter structures
- Kannada and English language switch
- Dark and light mode
- Compact AI constituency desk
- Admin publishing console concept at `/admin`
